## Setup

Download [rcm](https://github.com/thoughtbot/rcm)
`sudo emerge --ask app-admin/rcm`

Setup your `.rcrc` file. (run `man rcrc` to view an example)
`nvim ~/.rcrc`


See the output
`lsrc`

When happy with the output
`rcup`
