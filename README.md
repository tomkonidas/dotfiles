## Setup

Download && Install [rcm](https://github.com/thoughtbot/rcm)

> Gentoo

```bash
sudo emerge --ask app-admin/rcm
```

Setup your `.rcrc` file.

> run `man rcrc` to view an example on how to set it up

```bash
nvim ~/.rcrc
```


See the output

```bash
lsrc
```

When happy with the output

```bash
rcup
```
